The **PyGaze** project website (`raw/www.pygaze.org.html`) — an open-source
Python toolbox for creating eye-tracking experiments, supporting a range of
(mostly IR) eye trackers. It appears in the raw inbox as a general
eye-tracking-tooling reference. It was cited in an earlier draft's bibliography
but is **not** in the plan's §21 canonical list — cite author-year / by URL.

## Key facts

- Open-source experiment-building toolbox (stimulus presentation + tracker
  interfacing) for eye-tracking research; not a webcam gaze *estimator* itself
  and not adopted as a harness tracker arm.
- Background context for the [[gaze-estimation]] tooling landscape; contrast
  with the browser-native, webcam-only trackers GazePry actually uses
  ([[webgazer]], [[webeyetrack]], [[eyegestures]]).
- The two other HTML files once in `raw/` (ScienceDirect mirrors of the
  Gaze-tutor [1] and Thilderkvist papers) were deleted in commit 9185c26; this
  is the only remaining raw HTML source.
