Yang & Lu (Beihang) — *GazeLLM: a plug-and-play zero-shot LLM reasoning
framework for boosting gaze target detection*, **Visual Intelligence 2025** —
bibliography **[11]**. Part of the [[gaze-conditioned-ai]] cluster: uses an LLM
to improve gaze-target detection. Context for GazePry, not a security result.
(`raw/GazeLLM...-2025.pdf`)

## Key facts

- A **zero-shot, plug-and-play** LLM reasoning layer that boosts gaze-target
  detection (where a person is looking in a scene); peer-reviewed.
- Illustrates the growing utility case for capturing gaze — one more benign
  reason the stream gets collected, widening GazePry's attack surface.
