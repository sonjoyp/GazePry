## Key facts

- **Primary:** PETS / PoPETs — strong topical fit, rolling quarterly deadlines.
- **Reach:** USENIX Security / CCS / NDSS if the end-to-end cross-site linkage
  demo is strong.
- **Workshop hedge / early feedback:** WPES (co-located with CCS).
- **Companion:** SOUPS, if a consent user study is added.
- **Rough critical path** (calendar-bound by IRB + session separation): IRB
  4–8 wk → pilot 2–4 wk → collection 6–10 wk → analysis 4–6 wk → writing 4 wk;
  ≈ 6–8 months. Aim: a PoPETs cycle in H1 2027 or a USENIX Security 2027
  deadline. Verify exact dates against live CFPs.
