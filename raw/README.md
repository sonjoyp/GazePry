# raw/ — source inbox

Drop new source material here: papers (PDF/text), web clips, reviewer
comments, meeting notes, dataset descriptions. Then ask the agent to
**ingest** — it reads anything not yet recorded in `wiki/log.md` and folds it
into the wiki per `wiki/SCHEMA.md`.

Files are never modified or deleted by ingestion; this folder is the immutable
source-of-truth layer. Do **not** drop raw participant gaze data here.
